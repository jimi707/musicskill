# musicskill 🎵

**网易云音乐 · 每日歌单智能轮换器**

```
npx skills add jimi707/musicskill
```

在 Claude Code 中调用：`/musicskill`

---

## 它能做什么

每天自动给你的网易云歌单换一批新歌——**精准数量、过滤已听、推送到手机**。

- **智能选歌** — 分析你的歌单和听歌记录，只推你没听过的
- **数量精确** — 说20首就20首，不多不少
- **手机调风格** — 编辑 GitHub 上的 config.json，曲库/数量/时间随时改
- **自动推送** — 换完歌自动推送到你的手机 Bark

---

## 和官方推荐的区别

| | 网易云推荐 | musicskill |
|---|---|---|
| **算法** | 黑箱，不知道为啥推这些 | 曲库透明，知道每首歌从哪来 |
| **自定义** | 只能点"不感兴趣" | 一句话改风格，随时换曲库 |
| **过滤已听** | 偶尔推听过的 | 严格过滤你所有歌单+听歌记录 |
| **数量** | 固定30首，过了就没了 | 你说几首就几首，每天换 |
| **持续性** | 每日推荐次日消失 | 歌单固定，持续累积 |
| **隐私** | 数据上传云端分析 | 本地运行，Cookie用完即弃 |
| **手机控制** | ❌ | 手机上改 GitHub 配置即可 |

## 首次设置

### 1. 安装依赖

```bash
pip install pycryptodome
```

### 2. 创建网易云歌单

在网易云 App 中创建一个空白歌单，拿到歌单 ID（URL 里的数字）。

### 3. 安装 Skill 并设置

```bash
npx skills add jimi707/musicskill
```

### 4. 配置

在 Claude Code 中运行：

```
/musicskill 帮我设置每日推荐
```

按提示提供：
- 歌单 ID
- Bark Key（可选，用于推送）
- 网易云用户 ID
- 每天几点推、推几首

会自动提取你桌面端网易云的登录 Cookie，全程不存密码。

## 自定义

### 修改曲库/风格

编辑 `config.json`：
- `pick_count` — 每次推几首
- `schedule_time` — 每天几点执行
- `smart_recommend` — 开启/关闭智能过滤
- `song_pool` — 歌曲列表，可自由增删

### 手机上操作

1. 打开 GitHub App → jimi707/musicskill
2. 编辑 config.json → commit
3. GitHub Actions 自动触发，重新跑一轮

## 技术原理

- **Cookie 提取** — 从网易云桌面端本地加密数据库中解密，不存密码
- **WeAPI 加密** — 双层 AES + RSA，模拟官方客户端请求
- **分批操作** — 增删歌曲每次 ≤10 首，避免 API 静默失败
- **数量验证** — 操作后强制检查歌单数量，缺歌补、多歌删

---

## 安全

- 账号密码永不存储，仅运行时从本地解密 Cookie
- 所有操作仅限于你指定的那个歌单
- Cookie 用完即弃，不落盘
- 第一次使用后建议删除 GitHub Token
