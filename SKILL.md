# musicskill — 网易云每日歌单轮换引擎

## 架构总览

```
┌─────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────┐
│ Cookie提取   │───▶│ login_refresh│───▶│ 口味分析      │───▶│ 选歌引擎  │
│ (临时py解密)  │    │ (会话续期)    │    │ (歌单+听歌记录)│    │ (日期种子)│
└─────────────┘    └──────────────┘    └──────────────┘    └─────┬────┘
                                                                  │
┌─────────────┐    ┌──────────────┐    ┌──────────────┐          │
│ Bark推送     │◀───│ 验证+补齐     │◀───│ 分批添加新歌   │◀────────┘
│ (手机通知)   │    │ (确保COUNT首) │    │ (失败逐首重试) │
└─────────────┘    └──────────────┘    └──────────────┘
```

## 功能

- **智能过滤** — 遍历你的所有歌单+听歌记录，排除已听过
- **状态持久化** — state.json 记录每日推荐，避免7天内重复推荐
- **自动补齐** — 添加不足目标数时自动从曲库补选
- **指数退避重试** — API请求失败自动重试（最长等15秒）
- **结构化日志** — 同时输出到控制台和 logs/ 目录
- **退出码** — 0=成功 1=配置错 2=Cookie错 3=登录错 4=部分成功 5=完全失败
- **Dry-Run** — `--dry-run` 只预览不修改
- **Bark推送** — 完成后推送歌曲摘要到手机

## 前置依赖

```bash
cd musicskill && npm install
pip install pycryptodome cryptography
```

## 使用方式

### 第一步：配置

在 Claude Code 中执行 `/musicskill`，按提示提供：
- 网易云歌单 ID
- 用户 ID
- Bark Key（可选）
- 每天推几首（默认 20）

### 第二步：运行

```bash
# 正常执行
node auto_rotate.js

# 只预览不修改
node auto_rotate.js --dry-run

# 同时写日志文件
node auto_rotate.js --log-file
```

### 退出码说明

| 退出码 | 含义 | 处理方式 |
|--------|------|----------|
| 0      | 完全成功，歌单精准目标数 | 正常 |
| 1      | config.json 格式或字段错误 | 检查配置文件 |
| 2      | Cookie 提取失败 | 确认桌面端已登录 |
| 3      | API 登录/刷新失败 | 检查网络或 Cookie 有效性 |
| 4      | 部分成功，歌单不足目标数但 > 0 | 下次自动补齐 |
| 5      | 完全失败或未捕获异常 | 查看日志排查 |

## 技术实现

### Cookie 提取
写临时 `.py` 文件到磁盘执行，从 Chrome 加密 Cookie 数据库（AESGCM + DPAPI）解密。执行后立即删除。

### API 调用
通过 `NeteaseCloudMusicApi` 社区包封装请求，自带指数退避重试机制：
```
attempt 1 → 等待 1s → attempt 2 → 等待 2s → attempt 3 → 等待 4s → 放弃
```
风控拦截(-462)时额外等待 5s 后重试。

### 选歌算法
1. 用当天日期字符串转数字作为随机种子
2. 线性同余生成器做 Fisher-Yates shuffle
3. 优先级：排除已听过 → 排除近7天推荐 → 不足时放宽限制

### 状态持久化
`state.json` 保存：
- `lastDate` — 今日是否已执行过（防重复）
- `lastPicks` — 本次推荐的歌曲名
- `history` — 最近365天的历史记录

## 故障排除

| 现象 | 原因 | 解决 |
|------|------|------|
| `Cookie 提取失败` | 桌面端未登录或Cookie已过期 | 打开网易云桌面端，确认登录状态 |
| `-462 风控拦截` | 请求频率过高 | 自动重试（内置），如持续出现请等待10分钟 |
| `歌单不足20首` | 部分歌曲ID已失效 | 执行 `--dry-run` 排查无效ID，更新 config.json |
| `Bark推送失败` | Bark Key 无效或网络问题 | 检查 config.json 中 bark_key |

## 安全

- 账号密码永不存储
- 临时 Python 提取脚本执行后立即删除
- Cookie 仅存在于内存和 Node.js 变量中
- 所有操作限定于 `config.json` 中 `playlist_id` 指定的歌单
- 日志文件不包含完整 Cookie 值
